# Quizz Programação

Neste repositório realizo uma aplicação de um quizz de três perguntas, onde é possível visualizar as respostas corretas e quantos acertos o usuário teve, esse sistema é um projeto do curso [JavaScript do básico ao avançado](https://www.udemy.com/share/102Ha43@AeEbeQ-vJlSokGPD86JXjl9_MCOMpmQp5DElq9h75OBGAZ7w8wKDnupyvWs2W3Em-g==/ "Conferir Curso")  da plataforma **Udemy**, ministrado pelo instrutor **Matheus Battisti**.

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML5
- CSS3
- VsCode

---

## 🧠 Conceitos Aplicados

- **Estruturas de Dados Dinâmicas:** Organização das perguntas, alternativas e respostas corretas utilizando um array de objetos no JavaScript.
- **Renderização Dinâmica de Conteúdo:** Manipulação do DOM para limpar a pergunta anterior e inserir a nova questão sem precisar atualizar a página.
- **Lógica de Pontuação e Acumuladores:** Criação de variáveis de controle para contar a quantidade de respostas corretas fornecidas pelo usuário ao longo do jogo.
- **Validação de Respostas e Condicionais:** Verificação em tempo real se a alternativa clicada pelo usuário corresponde à resposta gabaritada no objeto da questão.
- **Feedback Visual Instantâneo:** Alteração de classes CSS via JavaScript para destacar as respostas corretas/incorretas e exibir a tela final de resultados com o score.
- **Controle de Fluxo e Estado:** Gerenciamento do índice atual do quiz para saber quando avançar para a próxima pergunta ou encerrar o jogo exibindo o resumo.
